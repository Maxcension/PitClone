# PitCourse

A 2D game made with Processing 3.0

## Pour commencer

En création dans le cadre d'un projet en ICN, PitCourse n'est pas fini. Il est donc possiblement buggué et pas complet.

## Prérequis

* Java 8

## Installation

Il suffit de lancer PitCourse.exe
